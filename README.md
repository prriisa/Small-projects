# Small Projects

This repository contains small frontend projects created using HTML, CSS, and JavaScript.
Each project focuses on improving core web development concepts and hands-on practice.

## Projects

### 1. Card Generator
A simple web project that takes user data from a form and generates a personalized card dynamically using JavaScript.

### 2. Email & Password Validator
A simple frontend project that validates email and password inputs based on predefined rules and shows appropriate error or success messages.

### 3. Dark / Light Theme Toggle (OS Aware)
A modern dark–light mode switcher that automatically detects the user’s OS theme and also allows manual theme switching using a toggle button.  

The selected theme is saved in localStorage, and user preference is given priority over system settings.


### Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
This repository is created for learning, practice, and improving frontend development skills.
More projects will be added regularly.
